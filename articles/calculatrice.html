🧮 Calculatrice JavaScript

Dans le cadre d'un petit challenge entre amis, j'ai développé une calculatrice en JavaScript, accompagnée d'une interface simple en HTML et CSS.

🎯 Objectifs

L'idée était de découvrir JavaScript, un langage orienté web, tout en pratiquant mes premières notions de développement frontend :

  • 👆 Gérer les événements utilisateur
  • 🔁 Interagir avec le DOM (Document Object Model)
  • 🎨 Styliser une interface responsive et agréable

🧠 Fonctionnalités

La calculatrice prend en charge les opérations classiques :

  • ➕ Addition
  • ➖ Soustraction
  • ✖️ Multiplication
  • ➗ Division
Le tout dans une interface claire et fluide ⚡
