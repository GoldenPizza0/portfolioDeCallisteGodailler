📘 Projet Web + Android : Gestion des Interventions

Ce projet fait partie de nos travaux de groupe de fin d'année. Il visait à développer un système complet de gestion des interventions pour une entreprise, composé de deux volets : une application web pour l'administrateur et une application Android pour les intervenants.

🌐 Partie Web (Admin)

💼 L'administrateur utilise une interface web lui permettant de :

  • 👥 Gérer les effectifs des salariés et des clients
  • 📄 Créer et associer des contrats selon les domaines d'activité des clients
  • 🧑‍🔧 Lier les contrats aux intervenants et commerciaux pour chaque mission
🔧 Développé en PHP avec une base de données MySQL

📱 Partie Android (Intervenants)

L'application mobile permet à chaque intervenant de :

  • 📆 Visualiser et gérer ses interventions
  • ✅ Mettre à jour le suivi de ses tâches
🔨 Développé avec Android Studio (Java)

🧠 Objectifs pédagogiques

Ce projet nous a permis de :

  • ⚙️ Consolider nos bases en back-end PHP
  • 📱 Découvrir le développement mobile Android
  • 🔄 Travailler en méthodologie Agile avec Git, GitHub et Jira
