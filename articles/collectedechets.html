🧹 Projet Web – Ramassage des poubelles

Ce projet fait partie de l’un des nombreux travaux de groupe. L’objectif était de réaliser un site web fonctionnel en équipe, avec des rôles utilisateurs distincts et des fonctionnalités spécifiques liées à un système de gestion de ramassage des poubelles 🗑️.

🔧 Fonctionnalités prévues

  • 👤 Connexion possible en tant que client ou administrateur
  • 📝 Possibilité de s'inscrire
  • 🗓️ Interface admin : gestion des tournées de ramassage avec plusieurs paramètres
  • 📬 Suivi des demandes de ramassage des clients
  • 📍 Interface client : saisie d'une demande de ramassage avec coordonnées

🎯 Objectif pédagogique

Le projet visait à nous familiariser avec le développement backend, notamment en PHP avec communication vers une base de données MySQL. Il nous a permis de pratiquer :

  • 💾 Intégration de données utilisateurs
  • 📤 Traitement des formulaires
  • 🔄 Gestion des rôles et des sessions
  • 📡 Structure CRUD de base

📎 Remarques

⚠️ Nous n’avions pas bien compris toutes les consignes, ce qui a entraîné un non-respect du cahier des charges. Ce projet reste néanmoins une bonne première expérience de collaboration backend avec base de données.
