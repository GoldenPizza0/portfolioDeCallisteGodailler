💼 Projet TTJ : Recrutement Web & Mobile

Ce projet fait partie de nos travaux de groupe réalisés en fin d'année. Il visait à créer une plateforme complète de mise en relation entre entreprises et chercheurs d'emploi, à travers deux applications : une application web pour les entreprises et une application mobile Android pour les candidats.

🌐 Partie Web (Entreprise)

👔 L'utilisateur est une entreprise disposant de postes à pourvoir. L'application web permet de :

  • 📋 Gérer ses offres d'emploi
  • 👀 Voir les candidatures reçues
  • ✅ Accepter un ou plusieurs candidats pour un poste
🛠️ Développée avec Symfony (PHP) et une base de données relationnelle

📱 Partie Android (Chercheur d'emploi)

👤 L'utilisateur est un chercheur d'emploi qui peut :

  • 🔍 Parcourir les offres disponibles
  • 📨 Postuler directement via l'application
  • 📬 Suivre le statut de ses candidatures (acceptée ou non)
🧰 Développée avec Android Studio (Java)

🎯 Objectifs pédagogiques

Ce projet nous a permis de :

  • 🧱 Consolider nos bases sur le framework Symfony
  • 📱 Approfondir notre pratique du développement mobile Android
  • ⚙️ Travailler en méthodologie Agile avec Git, GitHub et Jira
